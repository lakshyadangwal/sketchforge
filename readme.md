## вҡЎ What is SketchForge Pro?

SketchForge Pro is a **full-featured browser-based drawing application** built with pure HTML, CSS and Vanilla JavaScript. It packs professional features вҖ” layers with blend modes, mirror/symmetry drawing, pixel art mode, a physics-based gravity brush, speed replay timelapse, and a daily drawing challenge вҖ” all inside a **single file you can install on any device**.

Open it in a browser. Draw. Install it with one tap. Works forever вҖ” even offline.

### рҹ’§ Gravity Brush

A physics-based tool unique to SketchForge. Draw a stroke and it seeds dozens of paint drops that:
- Accelerate downward with simulated gravity
- Fade as they fall
- Create drip effects, watercolor bleeds, paint trails

Press `G` or select **Gravity** from the tool panel.
